# CRUD de Produtos em Python
Projeto simples de CRUD (Create, Read, Update, Delete) desenvolvido em Python, com o objetivo de praticar lógica de programação, manipulação de arquivos e organização de projetos.

# Funcionalidades
- Cadastrar produtos
- Listar produtos
- Atualizar produtos
- Remover produtos
- Persistência de dados em arquivo JSON

# Como executar o projeto
1. Clone o repositório: git clone https://github.com/neuzavitoria-dev/produtos-crud-python.git
2. Acesse a pasta do projeto: cd produtos-crud-python/backend
3. Execute o arquivo: python crud.py
